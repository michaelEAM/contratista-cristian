# Portafolio de Construcción

Este proyecto es un portafolio web sencillo para mostrar tu trabajo en pintura, colocación de baldosa y construcción.

## Cómo usar

1. **Abrir el sitio**
   - Abre `index.html` en tu navegador.

2. **Agregar tus imágenes**
   - Coloca tus fotos en la carpeta `images/`.
   - Reemplaza los nombres de archivo en el código HTML (`images/pintura1.jpg`, `images/baldosa1.jpg`, etc.) con los nombres de tus fotos.

3. **Actualizar los datos**
   - Cambia el texto de contacto (WhatsApp, correo, ubicación) en la sección de contacto.
   - Ajusta los servicios en la sección "Servicios" según lo que hagas.

## Cómo editar

- Edición rápida: abre `index.html` en un editor de texto (por ejemplo, Visual Studio Code). El diseño es responsive y funciona en móvil y escritorio.

## Estructura de archivos

- `index.html` - Página principal.
- `images/` - Carpeta donde colocar tus fotos.

¡Listo! Ahora tienes un portafolio básico para mostrar tus trabajos de construcción.
